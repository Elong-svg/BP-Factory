# 质量检查清单（v4.5.4 图表表格强制版）

## 🔥 v4.5.4 核心更新：图表表格数量强制执行

**问题现状**：各专家撰写时图表和表格数量严重不足，流程监管专家和质量审核官检查不严格

**解决方案**：
1. **明确最低数量标准** - 每章图表≥3 张、表格≥2 个
2. **流程监管专家实时监控** - 专家撰写过程中实时检查
3. **质量审核官一票否决** - 数量不足直接打回重做
4. **图表表格计数自动化** - 使用脚本自动统计，避免人工疏漏

## 关口 2：专家撰写完成检查（18 项强制检查，新增图表表格检查）

### 📊 图表数量检查（流程监管专家重点监控）
- [ ] **行业研究员章节**：至少 3 张图表（行业规模图、竞争格局图、趋势分析图）
- [ ] **财务分析师章节**：至少 3 张图表（营收趋势图、利润结构图、财务指标图）
- [ ] **战略分析师章节**：至少 2 张图表（SWOT 矩阵图、战略规划图）
- [ ] **全文章表总数**：≥8 张（不含执行摘要和参考文献）
- [ ] **图表类型多样性**：必须包含折线图、柱状图、饼图/环形图、矩阵图
- [ ] **图表嵌入位置**：所有图表必须在相关文本段落中直接嵌入，禁止单独成章
- [ ] **图表质量标准**：每张图表必须包含标题、坐标轴标签、数据标签、图例、来源标注

### 📋 表格数量检查（流程监管专家重点监控）
- [ ] **行业研究员章节**：至少 2 个表格（行业数据对比表、竞争对手对比表）
- [ ] **财务分析师章节**：至少 3 个表格（三大财务报表、财务指标表、同比分析表）
- [ ] **战略分析师章节**：至少 2 个表格（SWOT 分析表、战略规划表、投资评级表）
- [ ] **全文表格总数**：≥7 个（不含执行摘要和参考文献）
- [ ] **表格质量标准**：每个表格必须包含表头、数据行、单位标注、来源说明
- [ ] **表格类型多样性**：必须包含对比表、数据表、分析表、总结表

### 🚨 绝对禁止项（违反即失败）
- [ ] **专家内容完整性**：所有专家内容 100% 保留，无任何删减
- [ ] **无内容重写**：未添加/修改/删除任何专家内容  
- [ ] **无内容添加**：未在专家内容基础上添加任何分析、总结或评论
- [ ] **图表位置正确**：所有图表已在各章节中直接嵌入，无单独成章
- [ ] **图表数量不足**：任何章节图表数量低于上述标准
- [ ] **表格数量不足**：任何章节表格数量低于上述标准

### ✅ 强制执行项
- [ ] **直接拼接操作**：使用文件合并命令直接拼接，未进行内容审查
- [ ] **标题编号统一**：使用 1.1, 2.1, 3.1, 4.1, 5.1, 6.1 格式
- [ ] **数据来源移除**：文中所有数据来源标注已移除
- [ ] **参考文献添加**：文末有统一参考文献章节，按类型分组
- [ ] **图片路径正确**：所有图片路径指向 charts/目录
- [ ] **专业技能调用**：已调用 Word/PDF 文档生成技能
- [ ] **无硬编码输出**：未使用 python-docx 等冒充专业技能
- [ ] **质量评分达标**：整合后质量评分 >= 80 分
- [ ] **图表表格计数脚本已执行**：使用自动化脚本统计图表表格数量

## 关口 4：最终检查（20 项强制检查）

### 📊 基础指标检查（流程监管专家 + 质量审核官双重检查）
- [ ] **总字数 ≥20000 字**：专家原始内容总和达标
- [ ] **总图表 ≥10 张**：全部直接嵌入相关文本（**最低标准，必须达到**）
- [ ] **总表格 ≥8 个**：每章包含 2-3 个专业表格（**最低标准，必须达到**）
- [ ] **章节完整**：执行摘要 + 7 大章节 + 参考文献
- [ ] **图表分布均匀**：每章至少 2-3 张图表，不得有章节图表数量为 0
- [ ] **表格分布均匀**：每章至少 2-3 个表格，不得有章节表格数量为 0

### 📝 内容质量检查  
- [ ] **论文式写作**：无提纲式 1.2.3 列表，全部段落论述
- [ ] **三层论述结构**：What-Why-How 分析深度达标
- [ ] **主旨式标题**：无数据堆砌到标题
- [ ] **语言质量**：无 AI 味（Humanizer 已调用）
- [ ] **图表质量**：所有图表清晰、专业、信息完整
- [ ] **表格质量**：所有表格规范、数据准确、格式统一

### 🔍 数据与格式检查
- [ ] **数据一致性**：所有财务数据完全一致
- [ ] **参考文献完整**：文末参考文献格式规范、分类清晰
- [ ] **第七章完整**：包含投资评级、核心逻辑、风险提示、目标价位
- [ ] **格式规范**：自动生成目录、统一排版
- [ ] **无提示词泄露**：无"本段 3200 字"等内部指令
- [ ] **图表表格计数脚本已执行**：自动化统计并输出报告

## ⚠️ 最终警告

**任何检查项未通过都将导致：**
1. 质量审核官一票否决
2. 项目立即打回重做  
3. 流程监管专家启动惩罚机制

**图表表格数量不足特别处理：**
- 图表数量 < 10 张 → **直接打回重做，无需申辩**
- 表格数量 < 8 个 → **直接打回重做，无需申辩**
- 任何章节图表数量 = 0 → **该章节专家打回重做**
- 任何章节表格数量 = 0 → **该章节专家打回重做**

**记住：拼接 = 复制粘贴，不是重写！图表表格数量是硬性指标，没有商量余地！**
