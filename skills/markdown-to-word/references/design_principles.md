# Design Principles for Document Typography

**来源**：完全学习自 `minimax-docx` 技能的设计系统

## 核心设计原则

### 1. 留白与呼吸感（White Space & Breathing Room）

**原理**：人眼阅读时不是连续的，而是通过眼跳（saccades）在词群间跳跃。留白为这些注视点提供落地区域。

**研究结论**：
- **60-70% 内容覆盖率** 感觉舒适专业
- **80%+** 开始感觉密集和官僚
- **90%+** 感觉压抑，读者会无意识地跳过

**应用标准**：
```
页边距：上 2.54cm / 下 2.54cm / 左 3.17cm / 右 2.54cm
行距：1.15 倍（276 twips，基于 240 twips/行）
段后距：8pt（160 twips）
```

### 2. 对比与层次（Contrast & Hierarchy）

**原理**：通过字号、颜色、粗细的对比建立视觉层次，引导读者注意力。

**应用标准**：

| 元素 | 字号 | 颜色 | 粗细 | 用途 |
|------|------|------|------|------|
| H1 | 28pt | #1F3864 深蓝 | 粗体 | 文档标题 |
| H2 | 24pt | #1F3864 深蓝 | 粗体 | 章节标题 |
| H3 | 18pt | #1F3864 深蓝 | 粗体 | 小节标题 |
| 正文 | 11pt | #333333 深灰 | 常规 | 主体内容 |

### 3. 接近性与分组（Proximity & Grouping）

**原理**：相关的元素应该在空间上接近，不相关的应该分离。

**应用标准**：
```
标题与下属内容：段后 6-12pt（保持关联）
段落之间：段后 8pt（明确分离）
列表项之间：段后 8pt（保持连贯）
```

### 4. 对齐与网格（Alignment & Grid）

**原理**：对齐创造秩序感，减少认知负担。

**应用标准**：
- **标题**：左对齐（现代商务风格）或居中（学术论文）
- **正文**：左对齐（中文阅读习惯）
- **表格**：表头居中，数据左对齐
- **列表**：悬挂缩进 0.36 寸

### 5. 重复与一致性（Repetition & Consistency）

**原理**：相同的元素在整个文档中应该保持一致的样式，减少读者的认知负担。

**应用标准**：
- 所有一级标题：完全相同的字号、颜色、间距
- 所有表格：统一的三线表样式
- 所有列表：统一的缩进和符号

### 6. 视觉流与阅读路径（Visual Flow）

**原理**：读者的视线自然遵循 Z 型或 F 型路径移动。

**应用标准**：
- **重要信息**：放在页面顶部和左侧
- **层级结构**：通过标题层级引导视线
- **分页**：在章节结束处分页，避免内容断裂

## 颜色系统心理学

### 深蓝色（#1F3864 / #2F5496）

**心理效应**：
- 专业、权威、可靠
- 冷静、理性、商务
- 适合企业报告、投资分析

**使用场景**：
- 标题（建立权威感）
- 表格标题栏（强调结构）
- 强调元素（引导注意力）

### 深灰色（#333333）

**心理效应**：
- 中性、客观、易读
- 不抢眼，适合长文阅读
- 比纯黑（#000000）更柔和

**使用场景**：
- 正文（减少视觉疲劳）
- 次要信息

### 浅蓝色（#D9E2F3）

**心理效应**：
- 清新、科技、现代
- 不抢眼，适合背景

**使用场景**：
- 表格条纹行（提高可读性）
- 强调框背景

## 字体选择原则

### 英文字体

**Calibri Light（标题）**：
- 现代、简洁、专业
- 字重轻，适合大字号标题
- 与中文字体搭配和谐

**Calibri（正文）**：
- 易读性高
- 字重适中
- 商务文档标准字体

**Times New Roman（学术）**：
- 传统、学术、正式
- 适合学术论文、研究报告

### 中文字体

**黑体（标题）**：
- 现代、简洁、有力
- 适合大字号标题
- 与 Calibri Light 搭配完美

**微软雅黑（正文）**：
- 屏幕显示优化
- 易读性高
- 现代商务风格

**宋体（学术）**：
- 传统、正式
- 适合学术论文
- 与 Times New Roman 搭配

## 表格设计原则

### 三线表（学术标准）

**特点**：
- 只有顶线、底线、表头下线
- 无竖线，减少视觉干扰
- 表头蓝色背景 + 白色文字

**适用场景**：
- 学术论文
- 研究报告
- 财务数据

### 条纹表（商务标准）

**特点**：
- 完整边框
- 隔行条纹（浅蓝色）
- 表头深蓝色背景

**适用场景**：
- 商业计划书
- 投资分析报告
- 企业白皮书

## 页面布局黄金比例

### 企业报告（A4）

```
纸张：210×297mm
页边距：
  上：2.54cm (1 英寸)
  下：2.54cm (1 英寸)
  左：3.17cm (1.25 英寸) - 预留装订空间
  右：2.54cm (1 英寸)
行距：1.15 倍
内容覆盖率：60-70%
```

### 学术论文（Letter）

```
纸张：8.5×11 英寸
页边距：1 英寸四面
行距：2.0 倍（双倍行距）
段落：首行缩进 0.5 寸
内容覆盖率：65-75%
```

## 错误示例与修正

### ❌ 错误：行距过紧

```
行距：1.0 倍
问题：视线容易串行，阅读疲劳
修正：改为 1.15 倍或 1.5 倍
```

### ❌ 错误：页边距过窄

```
页边距：0.5 英寸
问题：内容压迫感强，显得廉价
修正：改为 1 英寸或以上
```

### ❌ 错误：标题颜色过多

```
H1: 蓝色
H2: 红色
H3: 绿色
问题：视觉混乱，缺乏统一性
修正：统一使用深蓝色系
```

### ✅ 正确：统一的设计语言

```
H1: #1F3864 深蓝，28pt，粗体
H2: #1F3864 深蓝，24pt，粗体
H3: #1F3864 深蓝，18pt，粗体
正文：#333333 深灰，11pt，常规
表格：#2F5496 蓝色表头，#D9E2F3 条纹行
```
