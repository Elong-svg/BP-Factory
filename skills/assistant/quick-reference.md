## 自动触发

### 任务完成后
```
✓ self-improving 反思
✓ memory 记录学习
✓ ontology 创建实体
✓ tiering 更新 HOT
```

### 用户纠正时
```
✓ improving 记录修正
✓ memory 记录 interaction
✓ 提升到 MEMORY.md（如果重要）
```

### Session 启动时
```
✓ system 恢复记忆
✓ improving review corrections
✓ ontology 加载图谱
```

## 三层架构

```
🔥 HOT - 当前任务（频繁更新）
🌡️ WARM - 用户偏好（稳定存储）
❄️ COLD - 长期归档（总结存储）
```

---

## 定期维护

- **每周**: 整理层级 + 周总结
- **每月**: 归档 COLD + 验证图谱
- **每季度**: 清理过时 + 优化检索
